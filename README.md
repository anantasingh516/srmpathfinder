# SRM KTR Campus PathFinder 📍

A clean, responsive web application designed for students and visitors to navigate the SRM Institute of Science and Technology (KTR Campus). This project implements the **Floyd-Warshall Algorithm** to solve the All-Pairs Shortest Path problem, ensuring users find the most efficient route between campus landmarks.

## 🚀 Features
- **Dynamic Pathfinding:** Uses Dynamic Programming to calculate the shortest distance between multiple campus points.
- **Brand-Aligned UI:** Designed with the official SRM IST Blue and Yellow color palette.
- **Walk-Time Estimation:** Calculates travel time based on a standard campus walking speed of 80 meters per minute (1.33 m/s).
- **Mobile Responsive:** Optimized for use on smartphones while walking across campus.

## 🛠️ Technical Specifications
- **Algorithm:** Floyd-Warshall
- **Time Complexity:** $\Theta(V^3)$
- **Frontend:** HTML5, CSS3 (Flexbox), Vanilla JavaScript
- **Data Source:** SRM KTR Campus Distance Matrix

## 📍 Locations Included
- Arch Gate
- University Building (UB)
- Tech Park
- Java Canteen
- SRM Hospital

## 📚 Academic Context
Developed as part of the **Design and Analysis of Algorithms (DAA)** course (CO-1). The application demonstrates the practical utility of graph-based algorithms in urban and campus navigation.

## 🖥️ How to Run
1. Clone the repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. Select your start and end points and click **Calculate Route**.
